## MANDATORY EVALUATION FRAMEWORK

Before any feature, change, or decision — evaluate all 20 dimensions:

| # | Dimension | Question |
|---|---|---|
| 1 | Business Value | Does this grow ARR, reduce churn, or protect the business? |
| 2 | User Value | Does this genuinely make a healthcare professional's life easier? |
| 3 | Healthcare Compliance | Is the compliance logic correct for every configured country? |
| 4 | Legal Impact | Does this create legal exposure? Is there a signed agreement needed? |
| 5 | Security Impact | Does this introduce a new attack surface? Is access controlled? |
| 6 | Privacy Impact | Does this touch PII? Is consent obtained? Is data minimized? |
| 7 | Database Impact | Does this require new tables, indexes, RLS policies, or migrations? |
| 8 | API Impact | Does this affect any API contract? Is versioning needed? |
| 9 | Infrastructure Impact | Does this change GCP resource requirements or costs? |
| 10 | Mobile Impact | Does this work on PWA, React Native, iOS, Android? |
| 11 | AI Impact | Does this use AI? Is it cost-controlled and explainable? |
| 12 | Cost Impact | What is the build cost, monthly run cost, and annual cost? |
| 13 | Revenue Impact | What is the revenue opportunity? Which plan tier does this unlock? |
| 14 | Sales Impact | Does this help close a deal or expand an existing account? |
| 15 | Marketing Impact | Does this create a story worth telling? Does it differentiate us? |
| 16 | Scalability Impact | Does this work at 100 users and at 100,000 users? |
| 17 | Global Expansion Impact | Is this configurable by country, or does it hardcode a single market? |
| 18 | Investor Impact | Does this increase enterprise value or support a funding narrative? |
| 19 | Enterprise Value Impact | Does this contribute to SOC 2, ISO 27001, or acquisition readiness? |
| 20 | Long-Term Sustainability | Is this maintainable by a small team at scale? |

## FINANCIAL GOVERNANCE

Before implementing any feature, estimate:

| Cost Type | Estimate Required |
|---|---|
| Build cost | Developer hours × effective hourly rate |
| Monthly run cost | GCP, Supabase, AI API, email, push — all services |
| Annual cost | Monthly × 12 + amortized build |
| AI cost | Tokens per call × calls per month × model price |
| Infrastructure cost | Cloud Run, Cloud Build, Storage, Secret Manager |
| Support cost | Expected support volume × cost per ticket |
| Scalability cost | Cost at 1K, 10K, 100K users |

Provide for every significant feature:
- ROI estimate
- Break-even timeline
- Revenue opportunity
- Cost reduction alternatives
- Lower-cost implementation options
